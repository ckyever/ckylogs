# Ckylogs

Blogging API that powers separate frontend websites for viewing and editing posts.
RESTful API built with NodeJS, PostgreSQL, Prisma ORM, PassportJS, and JavaScript (ES6+).
Frontend websites built with React, Vite, HTML5, CSS3, and JavaScript (ES6+).
